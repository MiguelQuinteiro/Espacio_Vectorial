# Espacio_Vectorial 📐🚀

Este repositorio contiene el motor de cálculo de **Álgebra Lineal y Geometría Vectorial**, desarrollado en Visual Basic 6.0 para el manejo de espacios n-dimensionales.

## 🚀 Descripción

`Espacio_Vectorial` es una biblioteca técnica diseñada para la manipulación abstracta y aplicada de vectores. El proyecto proporciona las herramientas necesarias para realizar operaciones fundamentales en espacios euclidianos, sirviendo como núcleo de cálculo para motores gráficos 3D, simulaciones físicas y el análisis de proyecciones matemáticas complejas (como la distribución de números primos en el ciclo unitario).

Es la base matemática que permite transformar conceptos abstractos de dirección, magnitud y sentido en datos computables y visualizables.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Arquitectura:** Biblioteca de clases (.cls) y módulos de funciones (.bas) especializados en cálculo vectorial.
- **Funcionalidades Clave:**
  - **Operaciones Básicas:** Suma, resta y producto por un escalar en cualquier dimensión.
  - **Productos Vectoriales:** Implementación de Producto Punto (Escalar) y Producto Cruz (Vectorial).
  - **Normalización y Magnitud:** Funciones para el cálculo de la norma de un vector y vectores unitarios.
  - **Transformaciones de Base:** Lógica para el cambio de sistemas de coordenadas y proyecciones ortogonales.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Clases de vectores (`clsVector`) y módulos de álgebra lineal.
- `/tests`: Scripts de validación de operaciones y teoremas vectoriales.
- `/docs`: Documentación sobre los algoritmos de proyección y matrices de transformación.
